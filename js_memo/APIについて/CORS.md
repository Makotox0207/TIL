- Cross Origin Resource Policy
- origin
  - プロトコル, ドメイン, ポート番号の3つを合わせたもの。(https://aaa.com:80) 
  - 同一オリジン...jsファイルとサーバーが同じアドレスにある
  - アドレスが違う場合はクロスオリジン
  - originが違うものはセキュリティに引っ掛かりアクセスできないので、サードパーティのサーバー側で設定する必要がある
