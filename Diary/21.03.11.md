## 学習したこと
[【初心者向け】テキストをtメソッドで記載して翻訳ファイルをあてる（ちょっとした変更にも強いビューの作成） - Qiita](https://qiita.com/tanutanu/items/9c23b1742097e9ddacdb)
* t(“”)で翻訳で翻訳ファイルに登録したものをその言語で表示できる。
```ruby
  status:
    article:
      unpublished: 非公開
      published: 公開

```

```ruby
= t(“status.article.published”)
# => 公開
= t(“status.article.unpublished”)
# => 非公開
```
